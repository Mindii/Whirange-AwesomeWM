# Key bindings
`Mod` Win Key

- `Print + Click` Screenshot
- `Mod + Shift + e` Exit Awesome
- `Mod + Ctrl + r` Restart Awesome
- `Mod + f` Fullscreen

- `Mod + Enter` Terminal
- `Mod + d` Run Menu
- `Mod + e` File Manager
- `Mod + m` Music on console
