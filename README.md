# Key bindings
`Mod` Win Key

- `Print + Click` Screenshot
- `Mod + Shift + e` Exit awesome
- `Mod + Ctrl + r` Restart awesome
- `Mod + f` Fullscreen
- `Mod + Arrows` Change focused window
- `Mod + Shift + Arrows` Move focused window

- `Mod + Enter` Open terminal
- `Mod + d` Open run menu
- `Mod + e` Open file manager
- `Mod + m` Open mocp
